JavaScript and jQuery

Your Local Weather

This is a project from the site FreeCodeCamp. With the user's permission, the system's geolocator tracks the user's location. Once this is received, they are relayed to FreeCodeCamp's provided weather API and the local weather is returned. The icon and background change based on the current weather conditions. The temperautre can also be switched between Celsius and Fahrenheit.

Wikipedia Viewer

This is the Wikipedia Viewer project from FreeCodeCamp. This project uses the Wikipedia API to retreive the Top 10 page results based on user input. The user can then click any individual result and be redirected to that articles page. When clicking on the top phrase, a random article page is presented to the user in a seperate tab.

Projects Using Frameworks

Open-Booked Disney

Open Booked was my final project while at The Iron Yard. Built with React, Open Booked calls data from multiple API's to provide the user with information such as attraction wait times, hotel price ranges, and restaurant menus. Using a Firebase Database, users can also login to leave and view comments about their experiences to gain better knowledge on the "Must-Do's" and the "Must-Avoids".
